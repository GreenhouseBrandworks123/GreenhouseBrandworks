# Greenhouse Brandworks Website

A modern React + Vite website for Greenhouse Brandworks, showcasing the company's branding, design, advertising, and digital marketing services.

## About Greenhouse Brandworks

Greenhouse Brandworks is a Bangalore-based creative agency specializing in branding, corporate communication, advertising, design, and digital experiences. With over 20 years of industry experience, the company helps businesses build strong brand identities through creative design, content, digital marketing, websites, print solutions, and integrated communication strategies.

## Getting Started

### Prerequisites

* Node.js (v18+ recommended)
* npm

### Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project:

```bash
cd frontend
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```

## Build for Production

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

## Deployment

This project is deployed using Vercel. Any changes pushed to the configured GitHub branch automatically trigger a new deployment.
